# `scalePopModel`

scalePopModel: scale times in a population model


## Description

scalePopModel: scale times in a population model


## Usage

```r
scalePopModel(mod, scale)
```


## Arguments

Argument      |Description
------------- |----------------
`mod`     |     A popmodel (usually read by readPopModel)
`scale`     |     Scaling factor


## Value

a new popmodel with scaled times
